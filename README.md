# Osintgram Lite (Educational Version)

This is a simplified and educational version of [Osintgram](https://github.com/Datalux/Osintgram), focused on learning OSINT techniques using Instagram data.

## Features
- Username-based reconnaissance
- Metadata extraction
- Screenshot automation (optional)
